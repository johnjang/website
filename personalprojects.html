Ledger API

I started this project to under more about Golang. Golang seems to be getting more and more popular so I decided to make a simple REST http API using Golang and sqlite3 as my database. The purpose of API is very personal. I wanted to have some sort of ledger to keep track of how much I was spending. This will automatically parse my credit/debit data and give me a comprehensive report depending on my choice of filter. Access to this API is firewalled off since it contains my personal information, but source code is available in github along with static demo.

RESTful API using golang with sqlite3 as my database

Handles and parses my personal bank usage information and applies filters

Depending on user requests, it will categorize and show my spendings
